# Project Scope

This repository is about score-level anomaly detection for business process data.

It includes:

- hybrid scoring code;
- a tiny runnable example;
- replay scripts;
- compact result summaries;
- notes about the relation between ADAMM and GAMA.

It does not include:

- raw event logs;
- full GAMA training runs;
- full ADAMM training runs;
- large model checkpoints;
- production deployment code.

Use the original ADAMM and GAMA repositories when you need their complete implementations.
