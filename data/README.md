# Data

Raw event logs are not stored in this repository.

For full experiments, use the original event-log sources and place the files in the path expected by your local runner.

Good starting points:

- https://www.processmining.org/event-data.html
- https://www.tf-pm.org/competitions-awards/bpi-challenge

The included replay does not need raw logs. It uses the prepared score table in `results/benchmark/`.
